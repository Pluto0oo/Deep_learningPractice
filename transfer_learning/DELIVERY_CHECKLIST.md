## 九、使用指南

### 9.1 快速开始

```bash
# 1. 进入项目目录
cd transfer_learning

# 2. 查看项目信息
python main.py info

# 3. 运行实验
python main.py finetune  # 运行微调实验
python main.py bert      # 查看BERT报告
```

### 9.2 环境配置

```bash
# 基础依赖
pip install torch torchvision numpy scipy

# BERT实验依赖
pip install transformers peft accelerate
```
